# Forge Foundry

Building software, sharing knowledge, and continuously learning.

Forge Foundry is an open-source software ecosystem focused on Linux, Rust, Artificial Intelligence, and developer productivity. Our mission is to create practical tools, platforms, and educational resources that empower developers, creators, and learners.

---

## Mission

Forge Foundry exists to:

* Build practical open-source software
* Advance Linux and Rust development
* Explore AI-assisted software engineering
* Share knowledge through real-world projects
* Foster a community of builders and learners

We believe the best way to learn is by building, documenting, and sharing the journey.

---

## Projects

### ForgeOS

An immutable Linux operating system built on Fedora Atomic technologies.

**Focus Areas**

* Development workstation experience
* Gaming and creator editions
* AI-assisted workflows
* Secure and reliable desktop computing
* Open-source tooling integration

---

### OpenForge

A modern open-source addon and package management platform.

**Goals**

* Addon discovery and management
* Dependency validation
* Multi-game support
* Cross-platform architecture
* Community-driven ecosystem

---

### Rust-GPT

AI-assisted developer tooling built with Rust.

**Goals**

* Developer productivity
* Code understanding
* Project analysis
* Workspace-aware AI assistance
* Open and extensible architecture

---

### Forge Welcome

A first-run experience and onboarding platform for ForgeOS.

**Goals**

* Simplified system setup
* Guided onboarding
* System status visibility
* Developer environment preparation
* Better user experience
